# coin-chart

> 국내거래소 기준의 코인차트를 정리해줄 프로젝트입니다.

![](https://github.com/devjin0617/coin-chart/blob/master/image/image.png?raw=true)

## 실행하기 위해서는,

``` bash
# 의존성 모듈들을 설치합니다.
npm install

# serve with hot reload at localhost:8080
# 핫리로드가 가능한 서버를 실행합니다. URL localhost:8080 입니다.
npm run dev

# 배포시에는 아래 명령어를 이용합니다.
npm run build
```
